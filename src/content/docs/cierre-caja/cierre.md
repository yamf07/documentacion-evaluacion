---
title: Cierre de Caja (Fin de Turno)
description: A reference page in my new Starlight docs site.
---

Al terminar su turno, debe realizar el corte de caja para cuadrar el dinero.

1. Vaya al menú "Opciones" en la esquina superior derecha.

2. Seleccione "Cierre de Turno".

3. El sistema mostrará el resumen de ventas (Efectivo en caja esperado).

4. Cuente el dinero físico y escríbalo en el campo "Monto Real".

5. Presione [CERRAR TURNO]. El sistema imprimirá el reporte de cierre.
