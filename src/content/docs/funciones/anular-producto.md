---
title: Anular un producto
description: A reference page in my new Starlight docs site.
---

Si el cliente decide no llevar un artículo ya escaneado:

1. Seleccione el producto en la lista de la pantalla (se resaltará en azul).

2. Presione el botón rojo [ELIMINAR ÍTEM] o la tecla SUPR.

3. El sistema pedirá confirmación. Haga clic en Sí.

:::caution[Importante!]
Para anular una venta completa después de haberla cobrado, se requiere la clave de un Supervisor.
:::
