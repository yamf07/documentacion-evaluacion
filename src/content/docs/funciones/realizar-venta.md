---
title: Cómo realizar una venta (Proceso Estándar)
description: A reference page in my new Starlight docs site.
---

Esta es la función que usará el 90% del tiempo.

1. Asegúrese de estar en la pestaña "Nueva Venta" (Menú Principal).

2. Escaneo: Pase el producto por el lector de código de barras o escriba el código manualmente en el campo "Código de Producto".

- El producto aparecerá automáticamente en la lista de compra con su precio.

3. Cantidad: Si el cliente lleva más de una unidad del mismo producto, presione la tecla F2 y cambie el número en la columna "Cant".

4. Finalizar: Una vez escaneados todos los productos, presione el botón verde [COBRAR] (o la tecla F5).

5. Seleccione el Método de Pago (Efectivo, Tarjeta, Transferencia).

6. Si es efectivo, ingrese el monto recibido; el sistema calculará el Cambio automáticamente.

7. Presione [CONFIRMAR] para imprimir el ticket.
