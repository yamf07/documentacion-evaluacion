---
title: Buscar un producto sin código
description: A reference page in my new Starlight docs site.
---

Si el código de barras está dañado:

1. Presione el botón [BUSCAR] (Lupa).

2. Escriba el nombre del producto (ej: "Leche 1L").

3. Seleccione el producto correcto de la lista y haga clic en [AGREGAR].
