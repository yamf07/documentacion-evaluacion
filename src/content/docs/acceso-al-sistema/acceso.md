title: Acceso al Sistema
description: A reference page in my new Starlight docs site.

Para garantizar la seguridad, cada cajero debe iniciar sesión con sus credenciales únicas.

1. Haga doble clic en el icono VentaFácil en el escritorio.

2. Verá la pantalla de bienvenida.

3. Ingrese su ID de Usuario y Contraseña.

4. Presione el botón [INGRESAR].

:::note[Nota de Seguridad]
Si introduce la contraseña incorrecta 3 veces, el sistema se bloqueará por 15 minutos. Si esto ocurre, contacte a su supervisor.
:::
